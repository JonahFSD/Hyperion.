## SAE Wins in Small Clusters, Loses in Large

When stratified by cluster size, SAE's advantage is size-dependent.

**Numbers:**
- Size 2: SAE 0.379 > SIC 0.240
- Size 51+: SAE 0.206 < SIC 0.301

**Supports:** H01 (partially; within-size bins SAE still beats SIC), but shows confound is real.

**Implication:** Size stratification in retrieval tests (T03-T04) is critical.
